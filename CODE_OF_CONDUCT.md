# Code of Conduct – EC2 Webapp with S3 Backend

This project is designed to help people learn how to deploy a web app using AWS EC2 and S3.  
I want this to be a positive and respectful space for everyone.

## How to Act
- Be respectful and constructive in comments or contributions.
- Ask questions freely without judgment.
- Keep discussions focused on this project and cloud development.

## What’s Not Okay
- Personal attacks, insults, or harassment.
- Sharing malicious code or sensitive info.
- Spamming the repo or discussions.
